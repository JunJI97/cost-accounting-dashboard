# NOA Cost Accounting 시연 스크립트

## 1. 문제 정의

5개 본부가 20개 프로젝트를 동시에 수행할 때, 인건비와 직접비는 비교적 명확히 귀속할 수 있지만 사무실 임대료, 서버 비용, 관리부서 급여 같은 공통비는 프로젝트별 손익을 왜곡할 수 있습니다.

NOA는 공통비 배부 기준을 바꿔가며 프로젝트별 수익성과 원가 동인을 빠르게 확인하는 관리회계 분석 도구입니다.

## 2. 대시보드

- 전체 매출, 총원가, 순이익, 평균 이익률 확인
- 프로젝트별 손익표 확인
- 원가 구성 Top 8 차트로 인건비, 직접비, 공통비 비중 비교

## 3. 데이터 관리

- 프로젝트 매출과 배부 가중치 수정
- 직원 시급 수정
- Time-sheet 투입 시간 수정
- 직접비와 공통비 추가/삭제/수정
- CSV/JSON으로 외부 데이터 가져오기/내보내기

## 4. 배부 엔진

- 투입 시간 기준 배부
- 인건비 기준 배부
- 매출 기준 배부
- 전사 공통비 풀과 본부별 공통비 풀 비교

## 5. What-if 시뮬레이션

- 특정 프로젝트에 추가 인력 투입
- 프로젝트 매출 증감 입력
- 공통비 증감 입력
- 전체 순이익 영향과 프로젝트 영향 확인
- 시나리오 저장 후 비교

## 6. 리포트

- 프로젝트별 손익표 확인
- 원가 동인 자동 요약
- 본부별 수익성 표 확인
- 손익표 CSV와 본부별 CSV 다운로드
